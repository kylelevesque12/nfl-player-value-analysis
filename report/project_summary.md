# Project Reports Index

This project measures NFL offensive player value, predicts how it carries into the
next season, and compares value to contract cost (QB/RB/WR/TE, 2016–2025). The
reports below are meant to be read as one connected argument, not as separate
artifacts. Start with the full report; the rest go deeper on individual pieces.

## Start here

- **[Final project report](final_project_report.md)** — the complete narrative:
  the value metric and its hidden flaw, the efficiency-vs-opportunity
  decomposition, the two-stage model it motivates, axis-aware uncertainty, the
  player-value deliverable, salary efficiency, and honest evaluation.

## The core analytical thread

- [Value decomposition](value_decomposition.md) — splits value into efficiency
  and opportunity and shows, with year-over-year persistence, why the two must be
  modeled separately. *This is the project's central insight.*
- [Two-stage value model](two_stage_value.md) — predicts opportunity and
  efficiency separately and recombines them. Tested honestly, it *lost* to a
  single model on accuracy; what it contributes is the asymmetric (role-driven vs
  efficiency-driven) prediction interval, validated at ~81% coverage.

## Reproducing everything

```bash
pip install -r requirements.txt
python scripts/run_pipeline.py
```

Individual stages run with `--steps` (for example `--steps decompose,two_stage`).
Reports in this folder are a mix of hand-written narrative (the final report, this
index, the salary and 2026 summaries) and reports generated by the pipeline (the
decomposition, two-stage, methodology, interpretation, context, advanced,
fantasy, and weekly-win reports); regenerate the latter by running the pipeline.
